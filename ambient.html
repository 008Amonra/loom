Example Study: What an AI Agent Can Build in One Session.
This page was designed and coded by opencode (an AI coding agent) in a single conversation. Eighteen parallel sound sources synthesized entirely with the Web Audio API — no samples, no external files, no dependencies. Volume sliders, mute buttons, presets, a randomizer, and a recording export function. The prompt was a short description of sounds: winter forest, distant chatter, car melodies, bells, gravel, cat, dog, boats, monks, rain, birds. All of it runs in your browser, zero install.
