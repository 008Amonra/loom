AI Email & Telegram Agent

Send emails, read inbox, chat via Telegram — all through your AI assistant. Your credentials stay encrypted. Your passphrase stays yours.

What it does

  • ✓ Send email from up to 3 Gmail accounts
  • ✓ Read recent inbox messages (IMAP)
  • ✓ Telegram bot for two-way messaging
  • ✓ AES-256 encrypted credentials
  • ✓ Passphrase entered blind — never stored

Requirements

  • • Node.js 22+
  • • opencode (or MCP-compatible client)
  • • Gmail account(s) with app passwords
  • • Telegram account (optional)

⬇ Download Beta Package

23 KB — includes scripts, MCP servers, guides, and config template

After downloading:

tar xzf email-agent-beta.tar.gz -C ~/
cd ~/email-agent && npm install
# then open credentials.html in your browser
